# Love Films
Demo: http://lovefilms.herokuapp.com/

Top Films and search author and film.

### Tech

* [Laravel 8]
* [Twitter Bootstrap]
* [jQuery]

### Installation

Install the dependencies and start the server.

```sh
$ cd love-films
$ npm install
$ composer install
$ php artisan serve
```

### Modify file
```sh
$ .env
Variable:
TMDB_TOKEN=
```

Get Token: https://www.themoviedb.org/
